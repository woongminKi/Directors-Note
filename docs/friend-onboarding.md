# Director's Note 빠른 시작 안내

카타르시스 연기학원 v1 파일럿용 — 약 10분 안에 첫 평가까지 보낼 수 있게 작성된 가이드입니다.

> 막히는 부분 / 어색한 화면 / "이거 왜 이렇게 동작하지" 싶은 부분은 그때그때 카톡으로 알려주세요. 빠르게 고쳐서 다시 안내드립니다.

---

## 1. 처음 로그인

1. 브라우저에서 보내드린 도메인 (예: `https://catharsis.directors-note.app` — 실제 URL 은 따로 안내) 으로 접속
2. **카카오로 로그인** 클릭
3. 카카오 동의 화면에서 닉네임 + 이메일 동의 → 확인
4. 정상 진입 시 **대시보드** 화면이 보입니다. (학생 0명, 평가 0건 — 시작 상태)

> 만약 `회원이 아닙니다` 페이지로 튕기면 카톡으로 알려주세요. 계정 매칭 작업 한 번 더 해야 합니다.

---

## 2. 학생 추가

상단 메뉴 → **학생** → **+ 학생 추가**

| 항목 | 입력 |
|------|------|
| 이름 | 학생 본명 |
| 구분 | 자유 텍스트. 예: `1년차`, `2년차`, `재수생`, `중3` |
| 부모 동의서 받음 | 동의서 받았으면 ON. 없으면 OFF (평가 시작 불가) |

저장 → 학생 목록에 추가됩니다.

> 부모 동의서가 없으면 **평가 시작하기** 버튼이 disabled 됩니다. 학원 자체 보관 양식(서면/디지털) 으로 받은 뒤 ON 으로 바꿔주세요. PIPA(개인정보보호법) 안전망입니다.

---

## 3. 첫 평가 시작 (Approach-A)

> v1 은 AI 영상 분석 없이 **코치 직접 입력** 방식입니다. 코치가 학생을 평가한 뒤 키워드 메모를 채우면, 시스템이 그 메모를 학부모용 편지(letter)로 자연스럽게 풀어 씁니다.

1. 학생 카드 클릭 → 학생 상세 페이지
2. **시작하기 (이번 달 평가)** 버튼
3. 코치 메모 폼 진입

폼 항목 (모두 자유 입력, 비워도 됨):
- **호흡/발성** — 예: "복부 호흡 안정, 고음 떨림 약간"
- **표정** — 예: "독백 도입부 감정 빌드업 좋음"
- **움직임** — 예: "무대 동선 자연스러움"
- **딕션** — 예: "자음 또렷, 모음 약함"
- **입시 완성도** — 예: "본방 70%, 클라이맥스 호흡 보강 필요"
- **추가 메모** — 자유

> 키워드 정도면 충분합니다. 완전한 문장 안 써도 됩니다 — 시스템이 알아서 다듬어 줍니다.

**제출** → 1분 안에 학부모 편지 초안 생성.

---

## 4. 편지 검토 + 편집 + 전송

생성된 편지를 화면에서 바로 검토 가능:
- 어색한 부분 직접 수정
- 한 단락 빼기 / 추가 가능
- 코치 어투에 맞게 다듬기

**편집 완료 → 승인** 버튼 클릭.

이어서 **학부모 링크 발송** 단계:
- 시스템이 학부모 전용 링크를 생성
- 그 링크를 학부모께 카톡/문자로 직접 전달

> 학부모는 로그인 없이 링크만 클릭하면 편지를 볼 수 있습니다. 보안은 토큰(랜덤 키) 으로 처리되어 있어 링크를 모르는 사람은 접근 불가.

---

## 5. 학부모가 보는 화면

링크 클릭 시 학부모가 보게 되는 것:
- 학생 이름 + 학년
- 평가 날짜
- 편지 본문 (1-2 단락, 자연스러운 정중체)

**보이지 않는 것:**
- 내부 등급 (A/B/C/D) — 코치/원장만 확인 가능
- AI 점수 / 분석 데이터 — 학부모에게 절대 노출 X (정책 P2)

---

## 6. 대시보드 — 매일 보게 될 화면

상단 메뉴 → **대시보드** 에 모이는 정보:
- **이번 달 평가** 진행 현황 (학생 수 대비 완료 / 검토 대기 / 발송 완료)
- **검토 대기 큐** — 편지 초안 생성됐는데 아직 발송 안 된 평가들
- **발송 완료** — 최근 10건
- **평가 시작 큐** — 이번 달 아직 평가 안 한 학생들

원장님 계정은 **소속 코치별 진행률 행** 도 함께 보입니다.

---

## 7. 자주 막히는 지점 + 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| 카카오 로그인 후 `회원이 아닙니다` 페이지 | `public.users` 매칭 안 됨 | 카톡으로 등록한 이메일 + Kakao 가입 이메일이 같은지 확인 |
| "동의서 필요" 라는 버튼이 disabled | 학생의 부모 동의서 OFF | 학생 편집 → 동의서 ON |
| 편지 생성 1분 넘게 안 끝남 | OpenAI API 일시 지연 | 새로고침 후 재시도, 안 되면 카톡 |
| 편지에 어색한 영어 / 어색한 어투 | LLM 생성 한계 | 직접 수정 → 승인. 자주 발생하면 카톡으로 케이스 알려주세요 |
| 학부모 링크가 안 열림 (404 또는 만료) | 링크 만료 (30일 default) | 같은 학생 평가 다시 시작 → 새 링크 발급 |
| 같은 학생 같은 날 두 번 평가 시도 | 시스템이 자동으로 막음 | 이미 만든 평가로 이동됨 — 거기서 이어서 작성 |

## 8. 어디로 연락

문제 발생 / UX 어색 / 기능 요청 — **카톡** 으로:
- 어떤 페이지에서
- 어떤 행동 했을 때
- 어떤 결과가 나왔는지 (가능하면 화면 캡쳐)

빠르면 같은 날, 늦어도 1-2일 안에 처리해드립니다.

---

## 첫 주 권장 사용 패턴

1. 학생 2-3명 먼저 등록 (편하게 입력해보기 위함)
2. 그 중 1명 첫 평가 → 편지 생성까지 한 번 끝까지 가보기
3. 발송은 보내기 전 본인 카톡으로 먼저 미리보기 권장 (편지 어투 익숙해질 때까지)
4. 익숙해지면 학원 전체 학생 등록 → 매 월말 평가 1-2주차에 평가 진행

대략 첫 1-2주는 사용 패턴 안정화 기간으로 보면 좋습니다. 이 기간 동안의 피드백이 v2 방향 결정에 가장 큰 영향을 줍니다.
