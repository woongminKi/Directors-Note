#### (b) `.claude/settings.local.json` — push 권한 룰 (선택)

이 머신에는 `Bash(git push origin main)` 권한이 추가돼 있어서 Claude 가 직접 push 가능. 다른 머신에서도 같은 편의 원하면:

```bash
mkdir -p .claude
cat > .claude/settings.local.json <<'EOF'
{
  "permissions": {
    "allow": [
      "Bash(git push origin main)"
    ]
  }
}
EOF
```

(`.gitignore` 가 `.claude/settings.local.json` 무시.)
