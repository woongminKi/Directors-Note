# LLM-judge 채점 루브릭 v1 (확정 — 코치 calibration 대기)

> **v1 확정 결정(2026-06-04):** 노래 없음 가정(발성=대사+면접) / 몸짓은 rawResponseJson 보관(스키마 무변경) / 4축 단순평균·가중치 동일 / 등급 산출 옵션 A. 상/중/하 실제 기준 calibration(코치 6개 직접 채점)은 확보되는 대로 밴드 문구에 반영.


Phase 1에서 `src/lib/evaluation/prompts/judge-rubric-v1.ts`로 들어갈 채점 기준의 설계 초안.
근거: downloads/ 6개 영상(상1·상2=A / 중1·중2=B / 하1·하2=C) 18프레임 직접 관찰 (2026-06-04).

## 0. 관찰 기반 설계 전제
- 고정 광각 풀바디 샷, 단독 연기자, 같은 연습실, 맨발/검은 옷. **얼굴 매우 작음.**
- 3파트: p1(0–90s) 자유연기 / p2(90–150s) 무용(노래 유무 미확인) / p3(150s~) 압박면접(정면 정지 Q&A).
- 정지 프레임으로는 상/중/하 구분 불가 → 실력차는 실행 품질(정밀도·표현·발성·정서)에 있음 → 동작+오디오로만 판별.
- 3대 원칙: ①표정=얼굴 미세표정 의존 금지(신체+음성으로 추론) ②발성=오디오 필수 ③몸짓=최대 가시 신호.

## 1. 판정기 컨텍스트 (프롬프트 상단 고정)
> 당신은 한국 연기예술 입시 전문 심사위원입니다. 아래 영상은 단일 지원자의 단독 실기 영상이며, **고정된 원거리 풀샷으로 촬영되어 얼굴이 작게 보입니다 — 얼굴 디테일이 안 보인다는 이유로 감점하지 마십시오.** 표정·정서는 신체 표현과 음성에서 추론하십시오. 영상은 3파트로 구성됩니다: (1) 0–90초 자유연기 (2) 90–150초 무용/노래 (3) 150초~ 압박면접. 각 파트를 보고/듣고 아래 4개 축을 0–10으로 채점하고, 축별 근거 1–2문장을 한국어로 제시하십시오. 이 평가는 **코치 전용**이며 학부모에게 노출되지 않습니다.

## 2. 4개 축 루브릭 (각 0–10)
점수 밴드: A=8–10 / B=6–7.5 / C=4–5.5 / D=0–3.5 (기존 `deriveGradeFromScores` 경계와 정렬).

### ① 발성 (vocal) — 주 신호: 오디오 (p1 대사 + p3 면접) ※v1: 노래 없음 가정, p2는 무용으로 간주
관찰: 발음 명료도, 발성 안정성(호흡·성량·울림), 톤의 강약·완급·속도 조절, 대사/응답 전달력.
- **A**: 발음 또렷, 호흡 안정, 성량·울림 충분. 감정에 따라 강약·완급 자유자재. 끝까지 전달력 유지.
- **B**: 대체로 명료·안정. 강약 변화 있으나 일부 구간 단조롭거나 흔들림.
- **C**: 전달은 되나 발음 뭉개짐/성량 부족/호흡 불안정이 잦음. 톤 변화 적음.
- **D**: 웅얼거림·성량 부족으로 전달 곤란, 호흡 무너짐.

### ② 표정·정서 (expression) — 얼굴 미세표정 의존 금지, 신체+음성 정서로 추론
관찰: 감정의 진정성/몰입, 정서 변화의 설득력, (가시 범위의) 표정·시선·신체로 드러나는 내면, 음성 정서.
- **A**: 정서가 진실되고 장면에 몰입. 감정 변화가 신체·음성으로 명확히 전달. 클리셰 아님.
- **B**: 감정 전달되나 일부 표면적이거나 정서 전환이 매끄럽지 못함.
- **C**: 감정이 단조롭거나 작위적. 몰입 끊김이 보임.
- **D**: 정서 전달 거의 없음/외운 것을 읊는 느낌.

### ③ 몸짓 (movement) — 최대 가시 신호 (p2 무용 + 전체 신체 사용)
관찰: 자세·정렬, 동작의 정밀도·통제력, 공간 활용, 무용 기술(유연성·균형·라인·리듬), 제스처의 의도성.
- **A**: 정렬·균형·라인 안정, 동작 통제력 높고 의도가 분명. 공간을 적극·정확히 사용. 무용 기술 숙련.
- **B**: 기본기 양호, 다이내믹 있으나 일부 동작 흔들림/마무리 불명확.
- **C**: 동작이 어색하거나 통제 부족, 균형 무너짐, 공간 활용 소극적.
- **D**: 자세·균형 불안정, 동작 의도 불명확.

### ④ 입시완성도 (examReadiness) — 종합 무대 완성도 + p3 면접
관찰: 무대 장악력·집중·일관성, 3파트 통합 완성도, 면접 응답의 논리·태도·자신감, 입시 본방 대비 준비 수준.
- **A**: 시작–끝 집중 유지, 무대 장악력 있음. 면접 응답 논리·태도 우수. 본방 가능 수준.
- **B**: 전반 안정적이나 일부 흔들림/완성도 부족. 본방 대비 70–80%.
- **C**: 기본 틀은 있으나 완성도 낮음/면접 응답 약함. 본방 대비 50–60%.
- **D**: 전반적으로 미완성, 기초 단계.

## 3. 점수 → 등급  ※v1 확정: 옵션 A
- **4축(발성·표정·몸짓·입시완성도) 단순평균 → 기존 경계(A≥7.25 / B≥5.75 / C≥4.25 / else D).** 가중치 동일(전부 1.0).
- judge는 참고로 holistic 등급도 함께 출력(calibration에서 4축평균 vs holistic 비교용, 등급 산출엔 4축평균 사용).

## 4. 판정기 규칙 (재현성·안전 — 정합성② 대응)
- `temperature=0`, structured JSON output 강제: `{vocal, expression, movement, examReadiness, rationale:{...}, holisticGrade?}`.
- 프롬프트·루브릭 **버전 고정**(`judge-rubric-v0.1`), (영상해시,모델버전,루브릭버전) 점수 캐시.
- 한국어 근거 필수, **코치 전용**(P2: 부모 letter 경로 무접촉).
- borderline(등급 경계 ±0.3)만 self-consistency 3회 다중표결.

## 5. 결정 현황
1. **상/중/하 실제 기준 calibration** — ⏳ 대기: 코치(원장 친구)가 6개를 본 루브릭으로 직접 채점 → ground-truth로 밴드 문구 보정 + judge 검증. (원장용 채점 가이드 문서 별도 발송: `2026-06-04 원장님 채점 가이드.md`)
2. **p2 노래 유무** — ✅ v1: 노래 없음 가정(무용 전용). 발성=p1 대사+p3 면접. (코치 채점 시 실제 확인되면 갱신)
3. **'몸짓' 스키마 영향** — ✅ v1: rawResponseJson에 보관, `ai_analyses` 스키마 무변경.
4. **가중치/등급 산출** — ✅ v1: 4축 단순평균(가중치 동일) + 기존 경계.
