# 2026-06-01 파비콘 적용

## 작업 내용
favicon_io로 생성한 파비콘 세트를 Next.js 15 App Router 프로젝트에 적용.

## 변경 사항
- `src/app/favicon.ico` — 기존 파일을 신규 아이콘으로 교체 (Next.js 자동 감지)
- `public/` — PNG 아이콘 세트 추가
  - `favicon-16x16.png`, `favicon-32x32.png` (브라우저 탭)
  - `apple-touch-icon.png` (iOS 홈 화면)
  - `android-chrome-192x192.png`, `android-chrome-512x512.png` (PWA / Android)
- `public/site.webmanifest` — 매니페스트 추가. 생성기가 비워둔 `name`/`short_name`을 `Director's Note`로 채움
- `src/app/layout.tsx` — `metadata`에 `icons`(icon 16/32, apple-touch) 및 `manifest` 추가

## 확인 방법
- `bun dev` 후 페이지 소스의 `<link rel="icon">`, `<link rel="manifest">` 확인
- 탭 아이콘은 캐시 영향이 있으므로 강력 새로고침 또는 시크릿 창에서 확인

## 업데이트 (재적용)
- favicon_io를 다시 생성하여 아이콘 6종 전체(favicon.ico + PNG 5종)를 새 디자인으로 교체
- 경로·메타데이터 구조는 동일하여 코드 변경 없이 에셋만 갱신
